Why I rebuilt this website

2019-12-31 • Tags: Miscellaneous

A similar website was deployed about 3 years ago when I was drafting the resume for my first internship. However, I haven't updated the website since then.

I always think of rebuilding this website, but I didn't get enough motivation. This has changed recently - the followings are the reasons why I decided to rebuild this website:

  1. New year resolution: I should write more. The old "personal website" was not a real personal website as it only had one Hello World post.
  2. My parents' laptop was very slow and they rejected my offer to buy them a new one. So I gave them my laptop and bought myself a new one. Since the source files of this website were on my old laptop, I realized that I can't update this site anymore by the usual means, although I didn't update at all in the last 3 years... (Updating from different computers is possible, though). When I tried to download the compiled files from my GitHub, I realized how empty my site was. I see it as a chance to restart this blog on my new laptop.
  3. I recently changed my GitHub username from Illithor to illithor, since it's hard to tell whether the first letter is an I or an L. An warning email was sent to me later automatically, because GitHub thought my custom domain that points to the old GitHub Page would need to update the DNS records to point to the new page. This is not an issue (the domain part of the URL is not case-sensitive), but this email forced me to go to my website and, again, saw how empty it was.

The reasons above are enough for me to rebuild this website. So here we go. I rebuilt it from scratch with hexo init and I hope this time my motivation to update this website will last longer than before.
